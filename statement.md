# Problem Statement: SpamGuard AI

## Problem
In the digital age, phishing attacks and spam messages are a primary vector for cyber threats. Users are bombarded with malicious links and scams via SMS and Email. Manually filtering these is inefficient and error-prone.

## Scope
SpamGuard is a machine-learning-based utility that:
1.  Ingests text data.
2.  Preprocesses and standardizes the text.
3.  Uses a Naive Bayes algorithm to classify messages as Spam or Ham.
4.  Provides a simple interface for real-time checking.

## Target Users
* Cybersecurity students.
* General users wanting to verify suspicious texts.

## High-Level Features
* **Model Training:** capability to learn from new datasets.
* **Persistence:** Saves the trained model for instant reuse.
* **Real-time Analysis:** Instant classification of user input.
